## Q.3 문자열 뒤집기
### 숫자가 0과 1로만 구성되어 있으므로 숫자 뭉치를 숫자 블럭으로 생각한다면, 0블럭과 1블럭의 갯수는 1개차이 또는 같아야 한다. 따라서 둘중 적은 갯수의 블럭의 숫자를 뒤집어 주면 된다.
